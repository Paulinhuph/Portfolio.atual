# Portfólio Pessoal — Paulo Henrique

- Este repositório contém o código do meu portfólio pessoal desenvolvido como parte do meu processo de aprendizado em desenvolvimento web e tecnologias voltadas à programação e inteligência artificial.
- O objetivo deste projeto é criar um ambiente digital onde apresento minhas habilidades, estudos, certificações e projetos na área de tecnologia.

## Sobre o projeto

- Este portfólio foi desenvolvido com o auxílio de ferramentas de inteligência artificial como forma de estudo prático sobre desenvolvimento, engenharia de prompts e construção de interfaces modernas.
- Além de servir como vitrine profissional, o projeto também representa um laboratório de aprendizado, onde experimento novas tecnologias, conceitos de design e boas práticas de desenvolvimento.

## Tecnologias utilizadas

HTML
CSS
JS

## Estrutura do portfólio

- O site apresenta as seguintes seções:

- Apresentação profissional e futurista

- Sobre mim

- Habilidades e tecnologias

- Certificados e formação

- Projetos

- Contato

- A proposta visual segue um estilo tecnológico e futurista, com foco em uma experiência elegante e moderna.

## Objetivo do projeto

- Este portfólio tem como finalidade:
- Consolidar conhecimentos em desenvolvimento web
- Praticar engenharia de prompts utilizando ferramentas de IA
- Construir presença profissional na área de tecnologia
- Documentar minha evolução como desenvolvedor
- Alocar certificados e habilidades técnicas
  
## Sobre mim
- Sou estudante de Análise e Desenvolvimento de Sistemas, com interesse em programação, inteligência artificial e desenvolvimento de soluções tecnológicas.
Atualmente estudo:
- Python aplicado à automação e inteligência artificial
- Engenharia de prompts para interação com modelos de IA
- SQL e fundamentos de banco de dados
- Desenvolvimento web com HTML e CSS

Meu foco é desenvolver pensamento computacional sólido e criar projetos que integrem software, dados e inteligência artificial.
